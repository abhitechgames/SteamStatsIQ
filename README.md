# Steam Stats IQ

[![chrome users](https://img.shields.io/chrome-web-store/users/pbhipeklokmcldebjabigdiklaeajiea?label=chrome%20users&style=for-the-badge&logo=googlechrome)](https://chrome.google.com/webstore/detail/EXTENSION_ID)  
[![license](https://img.shields.io/badge/license-BSD--3--Clause-blue?style=for-the-badge)](LICENSE)

**Steam Stats IQ** is an open-source Chrome extension that provides real-time insights into Steam game statistics. Designed for gamers, developers, and analysts, it delivers essential data to monitor and analyze Steam games effortlessly.

<img src="https://github.com/user-attachments/assets/c4313fa7-0f02-42d1-9e1b-7b8f1d743ce6" alt="ss-1" style="width: 25%; height: auto;">


## 🌟 Major Features

- 📊 **Real-Time Follower Counts**: View the number of followers for any Steam game.
- 👥 **Concurrent User Data (CCU)**: Track how many players are active in real-time.
- 💰 **Revenue Estimates**: Gain insights into a game's estimated revenue.
- ⏱️ **Average Playtime Insights**: Understand how long players engage with the game.
- 🛒 **Pricing and Sales Statistics**: Analyze price trends and sales performance.

## 🛠️ Development Setup

1. 📥 **Clone the Repository**:
   ```bash
   git clone https://github.com/abhitechgames/SteamStatsIQ.git
   ```
2. 🌐 Load the Extension in Chrome:
    - 🧭 Navigate to chrome://extensions/ in your browser.
    - 🛠️ Enable Developer Mode:
        Toggle the Developer Mode switch in the top-right corner.
    - 📂 Click Load Unpacked:

3. Select the folder containing the cloned repository.

The extension is now ready for development and testing!
